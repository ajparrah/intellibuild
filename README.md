# Intellibuild

## Description
This is a NestJS application that uses the OpenAI to generate response from a given prompt. The application is designed to be useful service to handle context of model.

### Features
- [x] Check orthography and grammar
- [ ] Generate text completions
- [ ] Generate audio from text
- [ ] Generate text from audio


### Example
`Orthography check`

![Orthography check](./docs/example-orthography-check.png)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
